# Гайдлайн по рецензированию: ДЗ 1

> Этот файл — шаблон. Замените содержимое на критерии оценки вашего первого домашнего задания.

## Как оставить рецензию

1. Откройте вкладку **Files changed** в этом PR.
2. Кликайте на строки файла и оставляйте **inline-комментарии** — минимум 2 предметных комментария обязательны для зачёта рецензии.
3. Когда закончите — нажмите **Submit review** → **Comment** (или **Request changes**, если нашли существенные ошибки).

---

## Критерии оценки

Оценивайте работу по следующим пунктам. Для каждого пункта оставьте inline-комментарий там, где обнаружили проблему (или похвалите, если всё сделано хорошо).

### 1. Правильность ответов (обязательно)

- [ ] Все теоретические вопросы разобраны
- [ ] Формулы записаны корректно (проверьте индексы, знаки, размерности)
- [ ] Выводы и рассуждения логически связны

### 2. Оформление (обязательно)

- [ ] Математические выражения оформлены в LaTeX (`$...$` или `$$...$$`)
- [ ] Текст структурирован (есть заголовки или нумерация по вопросам)

### 3. Глубина понимания (дополнительно)

- [ ] Автор объясняет *почему*, а не только пишет ответ
- [ ] Есть примеры или иллюстрации там, где это уместно

---

## Типичные ошибки

- Перепутаны строки и столбцы при умножении матриц
- Неправильный порядок применения правила дифференцирования
- Отсутствие транспонирования в градиентах
